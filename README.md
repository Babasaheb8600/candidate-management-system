# Candidate Management System

A Python command-line project built to manage job candidates.

## Features
- Add new candidates
- View all candidates
- Search candidate by name
- Check job eligibility
- Show statistics (total, average CGPA, highest, lowest)
- All data saved permanently to file

## Technologies Used
- Python 3
- File Handling
- Functions, Loops, Conditionals
- Lists, Dictionaries, Strings

## How to Run
```bash
python3 cms.py
```
